# Mini Event Platform (MERN)

## Features
- User Registration & Login (JWT)
- Create Events
- RSVP to Events
- Prevent Overbooking
- Logout & Route Protection

## Tech Stack
- MongoDB Atlas
- Express.js
- React.js
- Node.js

## Run Locally
### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm start
